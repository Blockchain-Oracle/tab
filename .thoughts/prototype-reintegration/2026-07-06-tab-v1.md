This synthesis resolves all of them into ONE contract: one payment lifecycle, one demo-page model, one webhook schema, one Leash pay-loop architecture, one revocation model, one cap-accounting rule, one set of enums/routes/tables, one labeling rule for spike-gated rows, and one executable grep gate.
